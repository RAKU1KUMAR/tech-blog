HTML Heading

To display the section heading title or subtitle we can use the HTML heading tags.In HTML 5 we have 6 heading tags start from <h1>up to <h6>,Where <h1>specify the largest heading and <h6> represent the smallest or sub heading. If the content is specified by heading tags, then it would be displayed large and bold as compared to other text content present on the wed-page. example

      <h1> First Heading </h1>
      <h2> Second Heading </h2>
      <h3> Third Heading </h3>
